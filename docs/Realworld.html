    get_articles(Tesla.Client.t(), [opt]) ::
  {:ok, Realworld.MultipleArticlesResponse.t()}
  | {:error, integer()}
  | {:error, Realworld.GenericErrorModel.t()}
  | {:error, any()}
when opt:
       {:tag, binary()}
       | {:author, binary()}
       | {:favorited, binary()}
       | {:limit, integer()}
       | {:offset, integer()}

    Get recent articles globally

    Get most recent articles globally. Use query parameters to filter results. Auth is optional

    Query parameters

    • tag: Filter by tag
    • author: Filter by author (username)
    • favorited: Filter by favorites of a user (username)
    • limit: Limit number of articles returned (default is 20)
    • offset: Offset/skip number of articles (default is 0)
